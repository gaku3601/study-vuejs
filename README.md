# study-vuejs
vue.js勉強用リポジトリ

## 01-environment
・vue-cliを使用し、vue.jsを動作させる環境を構築します。  

## 02-package
・packageにcssフレームワークのVuetify.jsを追加し、美麗なインターフェースを構築します。  

## 03-component
・componentの勉強として、親コンポーネントから子コンポーネントを呼び出す方法を習得します。  
・親コンポーネントから子コンポーネントへの値渡しの方法を習得します。  

## 04-databinding
・vue.jsの双方向データバインディングの概念について習得します。
